Release Changes

The following features have changed or improved.

  • Core Modules
    • You can now include ASK CLI build hooks when deploying your project. Add the "ask-hooks-directory" property to your abcConfig.json to specify a directory name relative to your project root, and the contents of that directory will be copied to the ASK "hooks" directory during deployment. Including a "build.sh" or "build.ps1" script in that directory will cause the ASK CLI to run your own custom build script instead of the default NPM build. For more information, see: ASK CLI Deploy Command
